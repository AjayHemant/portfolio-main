C Programming Basics

Master the fundamentals of C programming language and memory management

Introduction to C Programming

C is a powerful, low-level programming language developed by Dennis Ritchie in 1972. It's the foundation of many modern programming languages and operating systems, including Unix and Linux.

Why Learn C?

  • Foundation Knowledge: Understanding C helps you learn other languages
  • System Programming: Essential for operating systems and embedded systems
  • Memory Management: Direct control over memory allocation
  • Performance: Compiled code runs efficiently
  • Portability: C code can run on different platforms

Basic Syntax and Structure

#include <stdio.h>

int main() {
    printf("Hello, World!\n");
    return 0;
}

Key Components:

  • #include: Preprocessor directive to include libraries
  • stdio.h: Standard input/output library
  • main(): Entry point of every C program
  • printf(): Function to display output
  • return 0: Indicates successful program execution

Data Types and Variables

C provides several built-in data types:

int

Integer numbers (-2,147,483,648 to 2,147,483,647)

float

Floating-point numbers (6 decimal places)

double

Double-precision floating-point (15 decimal places)

char

Single characters or small integers

int age = 25;
float height = 5.9;
double pi = 3.14159265359;
char grade = 'A';

Control Structures

Conditional Statements

if (condition) {
    // code block
} else if (another_condition) {
    // code block
} else {
    // code block
}

Loops

// For loop
for (int i = 0; i < 10; i++) {
    printf("%d\n", i);
}

// While loop
int count = 0;
while (count < 5) {
    printf("Count: %d\n", count);
    count++;
}

Functions

Functions are reusable blocks of code that perform specific tasks:

// Function declaration
int add(int a, int b);

// Function definition
int add(int a, int b) {
    return a + b;
}

// Function call
int result = add(5, 3);

Pointers and Memory Management

Pointers are variables that store memory addresses:

int num = 42;
int *ptr = #  // ptr stores the address of num

printf("Value: %d\n", *ptr);  // Dereference to get value
printf("Address: %p\n", ptr); // Print address

Dynamic Memory Allocation

#include <stdlib.h>

int *arr = (int*)malloc(10 * sizeof(int));
// Use the array...
free(arr);  // Don't forget to free memory!

Arrays and Strings

Arrays

int numbers[5] = {1, 2, 3, 4, 5};
char letters[] = {'a', 'b', 'c', 'd', 'e'};

Strings

#include <string.h>

char str1[20] = "Hello";
char str2[] = "World";
strcat(str1, str2);  // Concatenate strings
