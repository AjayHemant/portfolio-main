# IME Question Bank

A web-based question bank for Industrial Management and Entrepreneurship (ME-501) course.

## Features

- **Recurring Topics**: View frequently asked topics with frequency indicators
- **Likely Questions**: See questions with probability ratings for upcoming exams
- **Common Question Types**: Organized by question patterns and types
- **User Tracking**: Firebase integration to track user logins (optional)

## Local Development

1. Clone the repository
2. Start a local HTTP server:
   ```bash
   python3 -m http.server 8000
   ```
3. Open http://localhost:8000 in your browser

## Technologies Used

- HTML5, CSS3, JavaScript (ES6 Modules)
- Firebase Firestore (for user tracking)
- Responsive Design

## Firebase Setup (Optional)

If you want to enable user login tracking, update the Firebase configuration in `firebase-config.js` with your own Firebase project credentials.
